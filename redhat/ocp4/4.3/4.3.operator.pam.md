# openshift 4.3 operator 

使用 process automation manager 来展示operator hub的功能

整个demo过程，包括
- 在operator hub中查找到 process automation manager 的operator
- 订阅这个operator
- 查看operator启动状态
- 创建一个CR，部署一个process automation manager实例
- 登录这个实例
- 删除这个实例
- 删除这个operator
